# Daily Zen Quote

A simple Nextcloud app which shows a Zen Quote of the Day. The quotes come from [ZenQuotes.io](https://zenquotes.io/) using the [featured daily quote API](https://docs.zenquotes.io/zenquotes-documentation/#call-today).

### Motivation

This is a personal hobby project which I am using to learn about Nextcloud app development and AI-assisted development. Significant portion of the code has been written by Claude Code.
